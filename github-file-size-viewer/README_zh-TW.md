# GitHub File Size Viewer (GitHub 文件/文件夾大小顯示工具)

本腳本通過 GitHub API 獲取文件和文件夾的大小，並將其顯示在 GitHub 倉庫頁面對應的文件/文件夾鏈接旁邊。  
支持遞歸計算文件夾大小，提供人性化的大小格式。

## 功能特點

-   顯示 GitHub 文件和文件夾的大小
-   自動遞歸計算文件夾總大小
-   在文件/文件夾鏈接旁邊以直觀方式顯示大小

## 使用方法

-   確保配置了有效的 GitHub 個人訪問令牌（`GITHUB_TOKEN`）[此處](https://github.com/settings/tokens/new?description=GitHub%20File%20Size%20Viewer&scopes=repo)生成新令牌，以避免 API 請求限制。
-   打開任意 GitHub 倉庫頁面，腳本會自動運行，並在文件/文件夾鏈接旁顯示大小。
