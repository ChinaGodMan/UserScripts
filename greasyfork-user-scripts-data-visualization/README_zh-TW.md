# GrreasyFork 用戶腳本數據可視化

<img height=6px width="100%" src="https://media.chatgptautorefresh.com/images/separators/gradient-aqua.png?latest">

## 描述

該用戶腳本使用 Chart.js 庫從 GreasyFork 上的用戶腳本頁面中提取和可視化腳本數據。它提供了以下功能：

-   **數據提取**: 從用戶的腳本頁面中提取安裝次數數據，包括總安裝次數和每日安裝次數。
-   **數據可視化**: 使用圖表（條形圖）展示每個腳本的總安裝次數和每日安裝次數。
-   **數據展示**: 在圖表下方顯示總安裝次數、每日安裝總數以及已發布腳本的數量。

<img height=6px width="100%" src="https://media.chatgptautorefresh.com/images/separators/gradient-aqua.png?latest">

## 功能

1. **獲取用戶數據**: 從用戶腳本頁面中提取相關數據。
2. **數據可視化**: 使用 Chart.js 繪製圖表來展示腳本的安裝數據。
3. **展示統計信息**: 在圖表下方顯示總的每日安裝次數、總安裝次數和已發布腳本的數量。

<img height=6px width="100%" src="https://media.chatgptautorefresh.com/images/separators/gradient-aqua.png?latest">

## 安裝

1. **安裝 Greasemonkey 或 Tampermonkey 插件**: 請確保您的瀏覽器已安裝 Greasemonkey 或 Tampermonkey 插件。
2. **添加腳本**:
    - 點擊上方綠色安裝按鈕進行安裝

<img height=6px width="100%" src="https://media.chatgptautorefresh.com/images/separators/gradient-aqua.png?latest">

## 使用

1. **訪問 GreasyFork 用戶腳本頁面**: 打開任意用戶的腳本頁面，例如 [https://greasyfork.org/users/1169082](https://greasyfork.org/users/1169082)。
2. **查看圖表和統計數據**: 腳本將在用戶的個人頁面加載後自動生成並展示數據圖表和統計信息。
3. **從網絡或者從當前頁面加載**: 腳本在代碼頂端提供了一個設置,当為`true`時,從當前網頁獲取,为`false`,向 GreasyFrok 下載用戶數據`JSON`

<img height=6px width="100%" src="https://media.chatgptautorefresh.com/images/separators/gradient-aqua.png?latest">
