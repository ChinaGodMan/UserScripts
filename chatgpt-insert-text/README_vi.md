ChatGPT Hộp đầu vào、Khu vực văn bản hoặc khu vực nội dung có thể chỉnh sửa，Nhanh chóng chèn văn bản cụ thể với các phím tắt đặt trước，Cải thiện hiệu quả đầu vào và kinh nghiệm chỉnh sửa。

## Chức năng chính

- **Ánh xạ khóa phím tắt**

    Bằng cách kết hợp các khóa（giống `Control+Alt+Period`、`Alt+Shift+Comma` Chờ đợi）Chèn các đoạn văn bản được xác định trước。


## Ví dụ khóa phím tắt

| phím tắt | Chèn nội dung văn bản |

|--------------------------|--------------------------------------------------------|

| Control + Alt + Period | Write concisely and continuously. |

| Alt + Shift + Period | Provide content concisely and continuously without segmentation. |

| Control + Alt + KeyC | Omit lines that start with `#` or empty lines from inside functions. |

| Alt + Shift + KeyG | Adapt the code to make the required changes. |

| Control + Shift + Equal | chèn Markdown Đánh dấu khối mã ` ``` ` |

（Xem ánh xạ khóa phím tắt hoàn chỉnh trong tập lệnh `keyMap` Sự vật）
