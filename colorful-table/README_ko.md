# 화려한 그라디언트 웹 페이지 테이블 스크립트를 아름답게합니다

이 스크립트는 웹 페이지의 모든 테이블에 색상 구배를 자동으로 적용합니다.，수동 클릭 또는 작업이 필요하지 않습니다，테이블 열의 숫자 크기에 따라 다른 배경색 생성，이것은 테이블의 시각적 효과를 향상시킵니다。

#### 기능적 특징

1. **그라디언트 색상을 자동으로 적용합니다**：스크립트는 페이지의 테이블을 자동으로 스캔합니다.，각 열의 데이터에 그라디언트 색상을 적용하십시오，사용자의 수동 트리거링이 필요하지 않습니다。
2. **데이터 중심의 그라디언트 색상**：각 열의 값의 최소 및 최대 값에 따라，색상 구배 효과를 생성합니다，값이 크거나 작습니다，배경색이 더 분명할수록。
3. **지속적인 테스트**：스크립트는 페이지를 1 초에 확인합니다，새 테이블이로드되는 경우，자동 그라디언트를 자동으로 적용합니다。
4. **그라디언트 컬러 구성**：그라디언트 컬러 기반HSL컬러 모델，스크립트의 조정 가능한 톤 범위（`HUE_RANGE`）、포화（`SATURATION`）、명도（`LIGHTNESS`）등.。

#### 주요 매개 변수

-   **HUE_RANGE**: 그라디언트 색상의 톤 범위를 설정하십시오，기본값은입니다120도，녹색에서 빨간색으로 그라디언트를 나타냅니다。
-   **SATURATION**: 색상의 채도를 제어합니다，기본값입니다80%，밝은 색상 효과를 제공합니다。
-   **LIGHTNESS**: 색상의 밝기를 제어하십시오，기본값은입니다88%，그라디언트를 부드럽고 눈부신 채로 만드십시오。

#### 핵심 논리

1. **extractFirstFloat**: 셀 텍스트에서 첫 번째 부동 소수점 번호를 추출하십시오，비 디지털 콘텐츠를 무시하십시오，숫자 부품 만 사용하여 색상 구배를 생성하십시오.。
2. **applyGradient**: 각 테이블 열의 최대 및 최소값을 자동으로 계산합니다.，그런 다음 단위 열의 값에 따라，해당 그라디언트 배경색을 생성합니다。
3. **initializeTables**: 페이지의 테이블을 정기적으로 확인하십시오，초기화되지 않은 테이블에 구배 색상을 자동으로 적용합니다。

#### 시나리오를 사용하십시오

이 스크립트는 웹 테이블이 포함 된 모든 시나리오에 적합합니다.，웹 테이블에 컬러 그라디언트를 자동으로 적용 할 수 있습니다，사용자가 데이터 차이를 신속하게 식별하도록 도와줍니다，테이블의 시각화를 향상시킵니다。
