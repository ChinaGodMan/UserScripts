## 脚本功能

对网页上的文字进行高亮显示，如果对你有帮助，可以随意修改使用。

## UI 说明

1. **第一栏：网页限制**
   - 可以指定在哪些网页上起作用，比如 `bilibili`, `baidu`, `bilibili.com`
   - 输入网址的一部分或者全部。每一个网址用空格隔开
   - 如果什么都不输入，那么就会作用到所有网站上

2. **第二栏：分类信息**
   - 鼠标移动到对应的关键字上会显示你输入的内容

3. **第三栏：关键字栏**
   - 输入你想要高亮的词，空格隔开
