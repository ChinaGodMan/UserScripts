# GitHub 按日期排序

此用户脚本将 GitHub 仓库中的文件自动按日期降序排列，方便查看最新更新的文件。

## 特性

- 页面加载时自动按更新日期排序文件。
- 适用于所有 GitHub 仓库。

## 使用
- 脚本将在页面加载时自动按日期排序文件，无需手动干预。
