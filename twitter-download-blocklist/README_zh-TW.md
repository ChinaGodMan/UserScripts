## 介紹

用於快速下載 Twitter 被屏蔽用戶列表的用戶腳本。它通過自動滾動並提取被屏蔽用戶的信息，生成一個包含賬號和用戶名的 `.tsv` 文件，方便您備份和管理被屏蔽的賬戶。

---

## 功能

**自動獲取屏蔽列表**：

- 腳本會自動遍歷屏蔽賬戶頁面的內容，並將賬號信息保存到本地。

**文件下載**：

- 獲取到的屏蔽賬戶信息會保存為 `.tsv` 格式，方便您在 Excel 或其他工具中查看和管理。

**按鈕操作**：

- 頁面右上角會生成一個 “Download Blocklist” 按鈕，點擊即可手動觸發下載。

---

## 使用步驟

**添加書籤**：

- 創建一個新的書籤，命名為 `GetBlocklist`,將以下鏈接複製到書籤的 URL 部分：或者你也可以直接訪問[https://x.com/settings/blocked/all#DOWNLOAD](https://x.com/settings/blocked/all#DOWNLOAD)
- 創建一個新的書籤，命名為 `GetMutedlist`, 將以下鏈接複製到書籤的 URL 部分：或者你也可以直接訪問[https://x.com/settings/muted/all#DOWNLOAD](https://x.com/settings/muted/all#DOWNLOAD)

**點擊書籤或按鈕**：

- 點擊您創建的書籤，或者直接使用頁面右上角生成的 “Download” 按鈕。

**等待下載**：

- 腳本會自動遍歷頁面的屏蔽賬戶列表，並在遍歷完成後彈出下載對話框。

---

## 文件格式

下載的文件為 `.tsv` 格式，內容如下：  
No. Account Name  
1 @example1 User Name 1  
2 @example2 User Name 2
