# TikTok 사용자 정보 추출 스크립트

이 스크립트는 사용됩니다 TikTok 사용자 페이지에서 사용자 정보를 추출하고 추가 사용자 페이지를 표시합니다.。사용자를 포함하여 ID、영역、언어、계정 생성 시간、친구의 수 등，또한 사용자 아바타 다운로드 기능도 제공합니다。

---

## 주요 기능

1. **사용자 이름 변경을 감지합니다**  
   모니터링으로 URL 및 사용자 데이터의 사용자 이름，변화가 있는지 여부를 감지하십시오，변경하면 페이지가 자동으로 새로 고침됩니다。

2. **사용자 정보를 추출하십시오**  
   페이지에서 `__UNIVERSAL_DATA_FOR_REHYDRATION__` 스크립트 추출 사용자 정보，포함하다：

    - 사용자 ID
    - 지역 및 국기
    - 언어와 언어 이름
    - 계정 생성 시간
    - 친구의 수
    - 판매자입니까?
    - 조직 계정입니까?
    - 사용자 이름 수정 시간
    - 별명 수정 시간

3. **사용자 아바타를 다운로드하십시오**  
   버튼을 제공하십시오，사용자의 아바타를 직접 다운로드하려면 클릭하십시오，타임 스탬프로 저장하십시오 `.jpg` 문서。

4. **동적 인터페이스 디스플레이**  
   사용자 페이지의 공유 제목 컨테이너에서，정보 상자를 동적으로 삽입하십시오，상세한 데이터를 추출했습니다。

---

## 사용 방법

1. **설치 스크립트**  
   브라우저 사용자 스크립트 관리 도구에 코드를 복사하십시오.（좋다 Tampermonkey）。

2. **열려 있는 TikTok 사용자 페이지**  
   예를 들어 `https://www.tiktok.com/@username`。

3. **사용자 데이터를 봅니다**  
   페이지가로드 된 후，추가 사용자 정보 상자가 자동으로 표시됩니다，딸깍 하는 소리“Download Profile Pic”버튼은 아바타를 다운로드 할 수 있습니다。

---

## 주목해야 할 것

- **페이지 구조 종속성**  
  이 스크립트는 다음과 같습니다 TikTok 현재 페이지 DOM 구조，좋다 TikTok 페이지 구조를 업데이트하십시오，스크립트를 수정해야 할 수도 있습니다。

- **데이터 소스**  
  출처 한 데이터 `__UNIVERSAL_DATA_FOR_REHYDRATION__` 스크립트 태그，이 태그가 존재하지 않는 경우，스크립트가 실행되지 않습니다。

- **아바타 다운로드 종속성**  
  기능 종속성을 다운로드하십시오 `GM_download`，지원이 필요합니다 Greasemonkey / Tampermonkey。

---

## 예

메시지 상자에는 다음 내용이 포함되어 있습니다：

- 사용자 ID
- 영역（국기와 함께）
- 언어（성명으로）
- 계정 생성 시간
- 친구의 수
- 판매자 및 조직 계정 상태
- 사용자 이름 및 별명 수정 시간
- 사용자 아바타 다운로드 버튼
