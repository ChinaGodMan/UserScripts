# TikTok 用戶信息提取腳本

此腳本用於在 TikTok 用戶頁面上提取用戶詳細信息並顯示額外的用戶數據。包括用戶 ID、地區、語言、賬戶創建時間、好友數等，同時提供下載用戶頭像的功能。

---

## 主要功能

1. **檢測用戶名變更**  
   通過監聽 URL 和用戶數據中的用戶名，檢測是否發生變更，如果變更則自動刷新頁面。

2. **提取用戶信息**  
   從頁面中的 `__UNIVERSAL_DATA_FOR_REHYDRATION__` 腳本提取用戶信息，包括：

    - 用戶 ID
    - 地區及國旗
    - 語言及語言名稱
    - 賬戶創建時間
    - 好友數量
    - 是否為賣家
    - 是否為組織賬戶
    - 用戶名修改時間
    - 暱稱修改時間

3. **下載用戶頭像**  
   提供一個按鈕，點擊後可直接下載用戶頭像，保存為帶時間戳的 `.jpg` 文件。

4. **動態界面顯示**  
   在用戶頁面的分享標題容器中，動態插入一個信息框，顯示提取的詳細數據。

---

## 使用方式

1. **安裝腳本**  
   將代碼複製到瀏覽器用戶腳本管理工具（如 Tampermonkey）。

2. **打開 TikTok 用戶頁面**  
   例如 `https://www.tiktok.com/@username`。

3. **查看用戶數據**  
   頁面加載完成後，會自動顯示額外的用戶信息框，點擊“Download Profile Pic”按鈕可以下載頭像。

---

## 注意事項

- **頁面結構依賴**  
  本腳本依賴 TikTok 當前的頁面 DOM 結構，若 TikTok 更新頁面結構，腳本可能需要修改。

- **數據來源**  
  數據來源於 `__UNIVERSAL_DATA_FOR_REHYDRATION__` 腳本標籤，若此標籤不存在，腳本將無法運行。

- **頭像下載依賴**  
  下載功能依賴 `GM_download`，需要支持 Greasemonkey / Tampermonkey。

---

## 示例

信息框包含以下內容：

- 用戶 ID
- 地區（帶國旗）
- 語言（帶全名）
- 賬戶創建時間
- 好友數
- 賣家與組織賬戶狀態
- 用戶名和暱稱修改時間
- 用戶頭像下載按鈕
