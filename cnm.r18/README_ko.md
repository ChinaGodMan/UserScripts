## 소개

연령 검증이 필요한 웹 사이트를 자동으로 처리하는 데 사용됩니다，페이지에서 확인 버튼을 클릭하거나 필요한 것을 자동으로 추가하여 URL 매개 변수，사용자가 연령 확인 페이지를 완벽하게 통과하도록 도와줍니다，탐색 경험을 향상시킵니다。

---

## 기능

### 1. 확인 버튼을 클릭하십시오（`verifyButton`）

웹 사이트 도메인 이름과 구체적으로 일치함으로써 CSS 선택자，페이지의 연령 확인 버튼을 자동으로 클릭하십시오.。

#### 핵심 기능

- **구성 규칙**
    - 사용 `config` 객체 정의는 웹 사이트 도메인 이름 및 해당 버튼 선택기를 지원합니다.。
    - 특정 웹 사이트의 동적 처리를 지원합니다（좋다 DLsite 그리고 animate-onlineshop.jp）。
- **다중 검증 처리**

    - 방문 할 때마다 확인 해야하는 일부 웹 사이트의 경우（좋다 DLsite 모바일 버전），확인 상태가 저장되지 않았습니다。

- **자동 클릭 로직**
    - 현재 웹 사이트의 도메인 이름과 경로 일치。
    - 일치하는 결과를 기반으로 해당 선택기를 찾고 자동으로 클릭하십시오.。
