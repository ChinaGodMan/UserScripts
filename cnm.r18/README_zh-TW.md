## 簡介

用於自動處理需要年齡驗證的網站，通過點擊頁面上的驗證按鈕或自動添加必要的 URL 參數，幫助用戶無縫通過年齡驗證頁面，提升瀏覽體驗。

---

## 功能

### 1. 驗證按鈕點擊（`verifyButton`）

通過匹配網站域名和特定的 CSS 選擇器，自動點擊頁面上的年齡驗證按鈕。

#### 核心功能

- **配置規則**
    - 使用 `config` 對象定義支持的網站域名及其對應的按鈕選擇器。
    - 支持特定網站的動態處理（如 DLsite 和 animate-onlineshop.jp）。
- **多次驗證處理**

    - 針對部分需要每次訪問都驗證的網站（如 DLsite 的移動端版本），不保存驗證狀態。

- **自動點擊邏輯**
    - 匹配當前網站的域名和路徑。
    - 根據匹配結果找到對應的選擇器並自動點擊。
