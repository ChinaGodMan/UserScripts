# GitHub RepoNotesHelper

这是一个可以给你 star 过的仓库添加笔记的的脚本,并且你备注的信息可以上传下载到 GitHub Gist

## 简介

曾经 Star 过的仓库是不是忘记了它们的用途？**GitHub RepoNotesHelper** 是一个为你量身定制的脚本，允许你为 Star 的仓库添加备注。更棒的是，你的备注可以上传到 **GitHub Gist**，随时备份和恢复，方便又安全！

---

## 功能

-   **添加备注**：为 Star 的 GitHub 仓库添加自定义备注，方便记忆。
-   **轻松管理**：备注直接嵌入到仓库页面，直观且高效。
-   **数据同步**：支持将备注上传到 GitHub Gist，随时下载恢复数据。
-   **无缝集成**：自动检测页面上的所有仓库并添加备注功能，无需手动干预。
