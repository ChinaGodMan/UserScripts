# Twitter(X)-Plus

Provides several useful enhancements：

- Hide some buttons
- Block users with one click
- Copy tweet link with one click
- Automatically stop video loop
- Automatically unmute video

**Gadgets that are convenient for you to use，The codes are all copied**
