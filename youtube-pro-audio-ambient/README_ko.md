# 🎧 YouTube Pro Audio Enhancer

하나를 위해 **YouTube 플레이어**다음이 만든 고급 오디오 향상 스크립트，기반으로 **Web Audio API**，한 번의 클릭으로 영상의 저주파 강도를 강화할 수 있습니다.、보컬 선명도 및 전반적인 음량，일반 비디오에 더욱 전문적인 청취 경험을 선사하세요。

---

## ✨ 특징

- 🔊 **저주파 강화（Sub Bass）**  
  사용 Low Shelf 필터 강화 약. 60Hz 저주파，음악에 더 깊은 깊이감을 부여하세요.。

- 🧹 **탁한 밴드 감소（Mud Cut）**  
  존재하다 250Hz 근처에서 컷을 만드세요.，답답한 느낌을 줄여보세요，소리를 더 깨끗하게 만들어라。

- 🎤 **보컬 선명도 향상（Presence）**  
  홍보하다 2.5kHz 존재감，보컬을 더욱 돋보이게 해보세요、더 명확하게。

- ✨ **고주파 밝기 향상（Brilliance）**  
  향상시키다 8kHz 위의 고주파수，전반적인 청취 경험을 더욱 투명하게 만드십시오.。

- 📈 **동적 압축（Compressor）**  
  균형 볼륨 역학，크고 작은 문제 감소，더욱 안정적인 청력。

- 🔉 **전반적인 이득 증가（Gain）**  
  명백한 왜곡 없이，전체 볼륨을 적당히 늘립니다.。

---

## 🖱 용법

- 존재하다 YouTube 플레이어의 왼쪽 하단에 새로운 것이 추가됩니다 **🎵 오디오 부스트 버튼**
- 버튼을 클릭하세요. **켜다 / 오디오 향상 끄기**
- 버튼이 파란색으로 바뀌어 현재 비디오에 개선 사항이 활성화되었음을 나타냅니다.

---

## 💾 상태 메모리

- 스크립트는 스위치 상태를 자동으로 기록합니다.（`localStorage`）
- 페이지를 새로 고치거나 동영상을 전환한 후，마지막 열린 상태를 자동으로 복원합니다.

---

## 🔧 기술적 구현

- 사용 `AudioContext` 오디오 처리 환경 만들기  
- 통과하다 `createMediaElementSource()` 인수하다 `<video>` 오디오 스트림  
- 여러 개 사용 `BiquadFilterNode` + `DynamicsCompressorNode` 오디오 처리 체인 구축  
- 의 도움으로 `MutationObserver` 그리고 `yt-navigate-finish` 이벤트，적응 YouTube ~의 SPA 페이지 전환

---

## ⚠ 주의할 점

- 초기 활성화가 필요할 수 있습니다. **영상을 재생한 후** 오디오 컨텍스트를 활성화하려면（브라우저 제한）
- 다른 장치 / 이어폰 / 스피커 효과는 다양합니다.
- 이 스크립트는 로컬에서만 적용됩니다.，오디오 데이터는 수정되거나 업로드되지 않습니다.

---

## 🚀 적용 가능한 시나리오

- 🎶 음악 MV / 라이브 공연
- 🎧 팟캐스트 / 회견 / 지도 시간
- 📺 볼륨이 너무 낮거나 음질이 보통입니다. YouTube 동영상
- 💻 외부 음질이 약한 노트북
