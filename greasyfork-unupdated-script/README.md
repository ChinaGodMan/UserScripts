## 功能说明

1. 此脚本用于检测页面中的脚本列表项
2. 如果某脚本的最后更新时间超过指定的天数，则对其进行高亮显示(未更新的脚本背景显示为浅红)。
3. 可以自定义检测的时间范围（以天为单位）。
