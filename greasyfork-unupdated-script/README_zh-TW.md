## 功能說明

1. 此腳本用於檢測頁面中的腳本列表項
2. 如果某腳本的最後更新時間超過指定的天數，則對其進行高亮顯示(未更新的腳本背景顯示為淺紅)。
3. 可以自定義檢測的時間範圍（以天為單位）。
