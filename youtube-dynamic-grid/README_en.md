## Overview

Dynamically adjust the number of videos displayed per line according to the width of the browser window，OptimizedYouTubeVideo grid layout。  
This script ensures that the layout is responsive，This optimizes screen space，Provide a better browsing experience。  
This script calculates the best number of videos（Between3arrive12between），and apply the necessaryCSScover。

> This script cannot be guaranteed to be valid for a long time。

## Start quickly

**Using scripts**：

- Access in a browser[YouTube](https://www.youtube.com/)。
- The script will automatically adjust the video grid according to the window size。
- Resize the browser window to see dynamic grid adaptation。
- existURLAdd afterward`?debug=1`（For example，`https://www.youtube.com/?debug=1`）Debug logs can be enabled。

## Function

- **Responsive mesh**：Adjust the number of videos displayed per line according to the window width（3arrive12）。
- **Seamless integration**：Applicable toYouTubeHome page and other video grid pages。
- **Efficient update**：useMutationObserverReal-time grid adjustment with event listener。
- **Rewind mechanism**：Includes polling and retry to ensure thatYouTubeDynamic load compatibility。

## How to use

- The script will be inYouTubepage（`*://www.youtube.com/*` and `*://youtube.com/*`）Automatically run on。
- No configuration required；The grid will automatically adjust according to the width of your browser window。
- For optimal performance，Please avoid running multiple grid modification scripts at the same time。

## parameter

- `videoWidth`: 340px（Thumbnail width，Applicable to3440x1440The screen displays about9A video）
- `margin`: 40px（Total margins on the left and right）
- `gap`: 10px（Thumbnail spacing）
