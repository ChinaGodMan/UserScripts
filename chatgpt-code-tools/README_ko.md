# 스크립팅 기능 및 구현

코드 블록 추가“코드를 저장하십시오”그리고“코드를 복사하십시오”단추，코드 스 니펫을 신속하게 저장하거나 복사합니다。

## 기능적 개요

### 1. **코드 블록을 감지합니다**

- 각 코드 블록에 대한 작업 버튼을 자동으로 추가합니다（반복적 인 추가를 피하십시오）。

### 2. **액션 버튼을 추가하십시오**

- **코드 버튼을 저장합니다**：

    - 코드 블록 기반 언어（클래스 이름별로 파일 확장을 추론합니다）。

    - 확장에 따라 동적으로 설정됩니다 MIME 유형。

    - 다운로드 링크를 만들고 다운로드를 트리거하십시오。

- **코드 버튼을 복사합니다**：

    - 코드 스 니펫을 클립 보드에 복사하십시오。

    - 상태 피드백을 제공합니다（좋다“성공적으로 복사하십시오”또는“복사 실패”）。

### 3. **동적 적응**

- 감시 장치 DOM 변화，새로 추가 된 코드 블록에 버튼을 자동으로 추가하십시오。

### 4. **버튼 스타일과 상호 작용**

- 버튼 스타일을 사용자 정의합니다（배경색、액자、마우스 호버 효과）。

- 상태 피드백 메커니즘：

    - 상태：파란색 버튼。

    - 직장에서：녹색 버튼은 상태를 표시합니다（좋다“절약...”또는“복사”）。

    - 오류 상태：빨간색 버튼에 오류 메시지가 표시됩니다。
