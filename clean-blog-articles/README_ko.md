# Clean Blog Articles

## 기능

- 페이지에서 광고 및 산만 요소를 삭제하십시오。
- 페이지 레이아웃을 조정하십시오，콘텐츠를보다 집중시킵니다。
- 여러 플랫폼을 지원합니다，포함하다 CSDN、블로그 파크、Reddit、너겟、Zhihu、Baeldung、Medium 그리고 Stack Overflow。
- 사용자 정의 사본 기능（접근성이 텍스트를 복사 할 수있는 경우）。
- 와이드 스크린 및 수직 스크린 장치의 레이아웃을 자동으로 조정하십시오。

## 지원되는 플랫폼

- **CSDN**：로그인 프롬프트를 제거하십시오、사이드 바、툴바 등。
- **블로그 파크**：주석 양식을 숨기십시오、사이드 바 등，컨텐츠 디스플레이를 최적화하십시오。
- **Reddit**：프로모션 링크와 사이드 바를 숨기십시오，주요 내용을 집중된 방식으로 표시하십시오。
- **너겟**：권장 영역을 제거하십시오、주석 섹션과 같은 간섭 요소。
- **Zhihu**：색상 스타일을 최적화합니다，오른쪽 사이드 바를 숨 깁니다、광고 등。
- **Baeldung**：최고 광고를 숨기십시오、사이드 바 등。
- **Medium**：회원 프로모션 게시물을 숨기십시오，코드 블록 디스플레이를 최적화합니다。
- **Stack Overflow**：사이드 바 및 하단 바닥 글을 제거하십시오。
