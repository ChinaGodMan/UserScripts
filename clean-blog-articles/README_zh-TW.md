# Clean Blog Articles

## 功能

- 刪除頁面中的廣告和乾擾元素。
- 調整頁面佈局，使內容更加集中。
- 支持多個平台，包括 CSDN、博客園、Reddit、掘金、知乎、Baeldung、Medium 和 Stack Overflow。
- 自定義復制功能（如允許無障礙複製文本）。
- 針對寬屏和豎屏設備自動調整佈局。

## 支持的平台

- **CSDN**：移除登錄提示、側邊欄、工具欄等。
- **博客園**：隱藏評論表單、側邊欄等，優化內容顯示。
- **Reddit**：隱藏推廣鏈接和側邊欄，集中顯示主要內容。
- **掘金**：移除推薦區、評論區等乾擾元素。
- **知乎**：優化顏色樣式，隱藏右側欄、廣告等。
- **Baeldung**：隱藏頂部廣告、側邊欄等。
- **Medium**：隱藏會員推廣條，優化代碼塊顯示。
- **Stack Overflow**：移除側邊欄和底部頁腳。
