# иҮӘе®ҡд№ү YouTube ж’ӯж”ҫеҷЁйҹійҮҸж»‘еқ—и„ҡжң¬

## з®Җд»Ӣ

YouTube зҡ„и§Ҷйў‘йЎөйқўе’Ң Shorts йЎөйқўжҸҗдҫӣиҮӘе®ҡд№үйҹійҮҸж»‘еқ—еҠҹиғҪпјҢдҪҝз”ЁжҲ·иғҪеӨҹйҖҡиҝҮжӣҙзӣҙи§Ӯзҡ„з•Ңйқўи°ғж•ҙйҹійҮҸгҖӮеҗҢж—¶ж”ҜжҢҒй”®зӣҳеҝ«жҚ·й”®ж“ҚдҪңпјҢжҸҗй«ҳйҹійҮҸжҺ§еҲ¶ж•ҲзҺҮгҖӮ

---

## еҠҹиғҪ

### 1. **и§Ҷйў‘йЎөйқўйҹійҮҸж»‘еқ—**

- й’ҲеҜ№ `/watch` йЎөйқўзҡ„и§Ҷйў‘ж’ӯж”ҫжҸҗдҫӣд»ҘдёӢеҠҹиғҪпјҡ
    - иҮӘе®ҡд№үйҹійҮҸж»‘еқ—пјҢж”ҜжҢҒжӣҙзІҫз»Ҷзҡ„йҹійҮҸи°ғж•ҙгҖӮ
    - е°ҶйҹійҮҸи°ғж•ҙеӯҳеӮЁеҲ° `localStorage`пјҢйЎөйқўеҲ·ж–°еҗҺдҝқжҢҒд№ӢеүҚзҡ„и®ҫзҪ®гҖӮ
    - ж”ҜжҢҒйҖҡиҝҮй”®зӣҳ `ArrowUp` е’Ң `ArrowDown` и°ғж•ҙйҹійҮҸпјҲ10% еўһйҮҸпјүгҖӮ

### 2. **Shorts йЎөйқўйҹійҮҸж»‘еқ—**

- й’ҲеҜ№ `/shorts` йЎөйқўзҡ„и§Ҷйў‘ж’ӯж”ҫжҸҗдҫӣд»ҘдёӢеҠҹиғҪпјҡ
    - иҮӘе®ҡд№үйҹійҮҸж»‘еқ—пјҢеёғеұҖдҪҚзҪ®дёҺйЎөйқўйЈҺж је…је®№гҖӮ
    - е°ҶйҹійҮҸи°ғж•ҙеӯҳеӮЁеҲ° `localStorage`пјҢйЎөйқўеҲ·ж–°еҗҺдҝқжҢҒд№ӢеүҚзҡ„и®ҫзҪ®гҖӮ
