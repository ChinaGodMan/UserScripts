# 사용자 정의하십시오 YouTube 플레이어 볼륨 슬라이더 스크립트

## 소개

YouTube 비디오 페이지 및 Shorts 이 페이지는 사용자 정의 볼륨 슬라이더 기능을 제공합니다，보다 직관적 인 인터페이스를 통해 사용자가 볼륨을 조정하도록 허용합니다.。키보드 바로 가기 작업을 지원합니다，볼륨 제어 효율을 향상시킵니다。

---

## 기능

### 1. **비디오 페이지 볼륨 슬라이더**

- ~에 맞서 `/watch` 페이지의 비디오 재생은 다음 기능을 제공합니다.：
    - 맞춤형 볼륨 슬라이더，더 미세한 볼륨 조정을 지원합니다。
    - 볼륨 조정을 저장하십시오 `localStorage`，페이지가 새로 고침 된 후 이전 설정을 유지하십시오。
    - 키보드를 통한 지원 `ArrowUp` 그리고 `ArrowDown` 볼륨을 조정하십시오（10% 증가）。

### 2. **Shorts 페이지 볼륨 슬라이더**

- ~에 맞서 `/shorts` 페이지의 비디오 재생은 다음 기능을 제공합니다.：
    - 맞춤형 볼륨 슬라이더，레이아웃 위치 및 페이지 스타일과 호환됩니다。
    - 볼륨 조정을 저장하십시오 `localStorage`，페이지가 새로 고침 된 후 이전 설정을 유지하십시오。
